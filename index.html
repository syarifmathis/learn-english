☁ GitHub Sync

Your notes, glossary edits, and progress will be saved as JSON commits to a file in your GitHub repo. Works across devices, keeps version history.

One-time setup

  1. Open github.com/settings/personal-access-tokens/new (fine-grained tokens).
  2. Token name: anything (e.g. sentence-practice). Expiration: as long as you want.
  3. Repository access → Only select repositories → pick the repo where you host this app.
  4. Permissions → Repository permissions → Contents: Read and write.
  5. Click Generate token, copy it, paste below.

Connection

Stored only in your browser's localStorage. Never sent anywhere except api.github.com.
